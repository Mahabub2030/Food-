## 🔧 Installation & Setup
To run this project locally, follow these steps:

### **For Linux & macOS:**
```sh
sudo git clone https://github.com/codewithsadee/foodie.git
cd foodie
open index.html
```

### **For Windows:**
```sh
git clone https://github.com/codewithsadee/foodie.git
cd foodie
start index.html
```
